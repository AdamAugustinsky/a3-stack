# Session Logs

Use session logs to capture decisions and outcomes so future reviews are grounded.

Location
- Store logs in `docs/ai/sessions/`.
- File name format: `YYYY-MM-DD-topic.md`.

Required Sections
- Goal
- Constraints
- Actions
- Outcomes
- Follow-ups

Optional Sections
- Phase Log (Explore, Plan, Implement, Review/Test)

Template
- Start from `docs/ai/sessions/_template.md`.

Optional Hook
- Enable the non-blocking reminder hook with `git config core.hooksPath .githooks`.
